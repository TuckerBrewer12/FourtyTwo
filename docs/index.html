How to Play

Texas 42 in a nutshell. Each hand has bidding, trump selection, 7 tricks, then scoring.

1
Deal & Bid
Each player draws 7 tiles. Starting left of the dealer, each player bids (min 30) or passes. Highest bidder wins and picks trump.
2
Declare Trump
The high bidder privately selects a suit (0–6). All 7 tiles of that suit outrank every other tile for the hand.
3
Play 7 Tricks
Leader plays any tile. Others must follow suit if able. Highest trump (or highest lead suit) wins the trick. Trick winner leads next.
4
Score the Hand
Count tricks (1 pt each) + count domino values. If bid team meets/beats their bid, both teams keep their points. Otherwise bid team scores 0.
5
Win the Game
First team to 250 points wins. If both cross 250 on the same hand, the team that made the bid wins.

Rules FAQ

Common questions about Texas 42.

What is the trump suit?
The player who wins the bidding round declares one suit (0–6) as trump. All 7 dominoes of that suit automatically beat any non-trump tile, regardless of the other number on the tile. The double of the trump suit is always the highest trump.
What does "follow suit" mean?
  • If the lead tile contains trump, everyone must play a trump tile if they have one.
  • Otherwise, players must play a tile with the same number as the highest end of the lead tile.
  • If you can't follow suit, play any tile — even trump.
What are "Low" and "42" bids?
Low (0) — Your team must capture no count dominoes. A tricky defensive bid.

42 — Slam bid. Your team must win all 42 points (all tricks + all count tiles).
What if everyone passes?
The dealer is forced to bid 30 and must play the hand. They choose trump and lead the first tile.
Which team is which?
Players sit across from their partner: P1 & P3 form Team 1, and P2 & P4 form Team 2. Partners' points combine each hand.
7-Mark variant?
In the simplified variant, winning a hand earns your team 1 mark. First team to 7 marks wins. Select this mode when creating a room.

Deploy & Play

GitHub Pages hosts this landing page. The multiplayer game needs a Python server — here's how to get it running.

⚠️ Why not GitHub Pages for the game? GitHub Pages serves only static files. Texas 42 requires a real-time WebSocket server (Python + Flask-SocketIO) for multiplayer communication. Run the server on any free Python host or locally — it takes under a minute.

▶ Run Locally (fastest)

# 1. Clone the repo git clone https://github.com/TuckerBrewer12/FourtyTwo.git cd FourtyTwo # 2. Install Python dependencies pip install flask flask-socketio eventlet # 3. Start the server python server/fourty_two_socketio_server.py # 4. Open in browser → http://localhost:5000
